getImage()

getImage(string  $email, integer  $size = 80, string  $default = 'mp', string  $rating = 'g', boolean  $htmlImg = false, array  $htmlImgAttributes = array()) : string

Get Gravatar image tag with URL or just URL.

Parameters

string $email

The email address of target Gravatar.
integer $size

Gravatar size.
string $default

Default image. Accept: 404, mp, identicon, monsterid, wavatar, retro, robohash, blank.
string $rating

Rating. Accept: g, pg, r, x.
boolean $htmlImg

Set to true to return &lt;img&gt; tag. Default is false means it will be return only URL.
array $htmlImgAttributes

The associative array of img attributes.

string —

Return URL or img tag with URL.
